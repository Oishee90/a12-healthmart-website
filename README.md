# HealthMart

HealthMart is a multi-vendor e-commerce platform specializing in the sale of medicines and healthcare products. The platform provides a seamless user experience for purchasing medications, submitting queries, and receiving expert advice using the MERN stack.



## Live Site URL
[HealthMart Live Site](https://medicine-selling-website.web.app)

## Features
- **Responsive Design**: The website is fully responsive and works seamlessly on mobile, tablet, and desktop devices.
- **User Authentication**: Sign up, log in, and social login (Google/GitHub) with role selection (user or seller).
- **Product Management**: Users can browse, view, and purchase medicines. Admins and sellers can manage products through their dashboards.
- **Real-time Updates**: Real-time notifications for CRUD operations, successful authentication, and other user actions using toast notifications.
- **Admin Dashboard**: Manage users, categories, payments, sales reports, and banner advertisements.
- **Seller Dashboard**: Manage medicines, view payment history, and request advertisement slots.
- **User Dashboard**: View payment history and track orders.
- **Category Management**: Users can browse medicines by categories with detailed information.
- **Shopping Cart and Checkout**: Users can add medicines to the cart, update quantities, and proceed to checkout with Stripe payment integration.
- **Invoice Generation**: Users receive a detailed invoice after purchase, which can be downloaded as a PDF.
- **Advanced Filtering**: Pagination, sorting, and search functionality for medicine tables.
- **Environment Variables**: Sensitive information such as Firebase config keys and MongoDB credentials are hidden using environment variables.
